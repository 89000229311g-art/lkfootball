# ⚽ Тестирование Football Academy System

## 🎯 БЫСТРЫЙ СТАРТ

### 1️⃣ Установка зависимостей (первый раз)
```bash
cd "/Users/macbook/Desktop/football-academy-system 2"

# Установите Python зависимости
pip3 install -r requirements.txt

# Или с виртуальным окружением
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 2️⃣ Создание тестовых пользователей
```bash
python3 create_users.py
```

### 3️⃣ Запуск Backend
```bash
uvicorn app.main:app --reload
```
Backend будет доступен на: http://localhost:8000

### 4️⃣ Запуск Frontend (в новом терминале)
```bash
cd frontend
npm install  # только первый раз
npm run dev
```
Web приложение будет на: http://localhost:3000

### 5️⃣ Запуск Mobile App (в новом терминале)
```bash
cd football_academy_app
flutter pub get  # только первый раз
flutter run
```

## 📱 МОБИЛЬНОЕ ПРИЛОЖЕНИЕ - Тестирование

### Для Android Emulator:
1. Запустите Android Studio
2. Откройте AVD Manager
3. Запустите эмулятор
4. Выполните: `cd football_academy_app && flutter run`

### Для iOS Simulator:
1. Запустите Xcode
2. Откройте Simulator
3. Выполните: `cd football_academy_app && flutter run`

### Быстрый вход:
- На экране входа есть **4 кнопки** для быстрого выбора роли
- Нажмите на кнопку (Owner/Admin/Coach/Parent)
- Логин и пароль заполнятся автоматически
- Нажмите "Войти"

## 🐛 РЕШЕНИЕ ПРОБЛЕМ

### Backend не запускается:
```bash
# Установите зависимости
pip3 install -r requirements.txt --upgrade

# Проверьте PostgreSQL
ps aux | grep postgres
```

### Frontend не работает:
```bash
cd frontend
npm install
npm run dev
```

### Mobile app не подключается:
- **Android Emulator**: Проверьте, что backend на порту 8000
- **iOS Simulator**: Используйте `localhost:8000`
- **Real Device**: Измените IP в `api_config.dart`
