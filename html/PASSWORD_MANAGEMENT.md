# Быстрая Инструкция: Управление Паролями

## Для Руководителей (super_admin)

### Изменить пароль администратору
```bash
PUT /api/v1/auth/users/{user_id}/password
Body: { "new_password": "newpass123" }
```

**Пример в коде:**
```python
response = await api_service.resetUserPassword(
    user_id=5,
    new_password="newpass123"
)
```

---

## Для Администраторов (admin)

### Изменить пароль тренеру
```bash
PUT /api/v1/auth/users/{user_id}/password
Body: { "new_password": "newpass123" }
```

### Изменить пароль родителю
```bash
PUT /api/v1/auth/users/{user_id}/password
Body: { "new_password": "newpass123" }
```

**⚠️ Нельзя:**
- Менять пароль другим администраторам
- Менять пароль руководителям

## API Endpoints

### 🔐 Сменить пароль пользователя
```
PUT /api/v1/auth/users/{user_id}/password
Content-Type: application/x-www-form-urlencoded

new_password=your_new_password
```

### 📝 Редактировать пользователя
```
PUT /api/v1/auth/users/{user_id}
Content-Type: application/json

{
  "full_name": "Новое Имя",
  "phone": "+373xxxxxxxx"
}
```

### ❌ Удалить пользователя
```
DELETE /api/v1/auth/users/{user_id}
```

---

## Коды Ошибок

| Код | Сообщение | Причина |
|-----|-----------|---------|
| 403 | "Администратор не может изменять пароль Руководителя" | Админ пытается трогать super_admin |
| 403 | "Администратор не может изменять пароль другим Администраторам" | Админ пытается трогать другого админа |
| 400 | "Для изменения своего пароля используйте /auth/me/password" | Попытка изменить свой пароль через административный эндпоинт |
| 400 | "Пароль должен содержать минимум 6 символов" | Слишком короткий пароль |

---

## Важно! 🚨

1. **Свой пароль** меняется через `/auth/me/password` (требует текущий пароль)
2. **Чужие пароли** меняются через `/auth/users/{id}/password` (только руководитель/админ)
3. Пароль должен быть **минимум 6 символов**
4. Нельзя изменить пароль пользователя **выше** вас в иерархии
