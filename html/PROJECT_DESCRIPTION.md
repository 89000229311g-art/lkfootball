# ⚽ Sunny Football Academy - Система управления футбольной академией

## 📋 Описание проекта

Комплексная система для управления детской футбольной академией. Включает веб-панель администратора, мобильное приложение для родителей и тренеров, а также REST API бэкенд.

---

## 🛠 Технологический стек

### Backend
- **Python 3.9+** - основной язык
- **FastAPI** - веб-фреймворк
- **SQLAlchemy** - ORM для работы с БД
- **SQLite/PostgreSQL** - база данных
- **Alembic** - миграции БД
- **JWT** - аутентификация
- **Pydantic** - валидация данных

### Frontend (Web)
- **React 19** - UI библиотека
- **Vite** - сборщик
- **Tailwind CSS** - стили
- **Axios** - HTTP клиент
- **React Router** - маршрутизация

### Mobile App
- **Flutter** - кроссплатформенное приложение
- **Dart** - язык программирования

### DevOps
- **Docker** - контейнеризация
- **Docker Compose** - оркестрация
- **Nginx** - reverse proxy

## 🚀 Запуск проекта

### Локальный запуск

#### Backend
```bash
cd football-academy-system
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

#### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Docker запуск
```bash
docker-compose up -d db backend
cd frontend && npm run dev
```

1. База данных SQLite используется для разработки, PostgreSQL для production
2. Переводы поддерживают: Русский, English, Română
3. JWT токены действительны 8 дней
4. Все пароли хешируются с использованием bcrypt
